# Task-02: Image Generation with Pre-trained Models

## Description
This project demonstrates text-to-image generation using a pre-trained
Stable Diffusion model.

## Model Used
- Stable Diffusion v1.5 (Hugging Face)

## Steps
1. Install dependencies
2. Load pre-trained model
3. Generate image using text prompt
4. Save output image

## Sample Prompt
"A cute robot reading a book in a library"

## Output
Generated images are saved locally.
